# Geolink

Монорепозиторий приложения Geolink.

## Структура проекта

| Папка | Описание |
|-------|----------|
| `Geolink.API/` | Backend API на .NET 9 |
| `Geolink.Mobile/` | Мобильное приложение на Flutter |

## Geolink.API

ASP.NET Core Web API.

### Запуск

```bash
cd Geolink.API
dotnet run
```

## Geolink.Mobile

Flutter мобильное приложение для iOS и Android.

### Требования

- Flutter SDK 3.x
- Dart SDK 3.x

### Запуск

```bash
cd Geolink.Mobile
flutter pub get
flutter run
```
